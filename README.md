# react-practice-projects
This repository contains multiple small React projects for practice:


- **Todo List**: Add, delete, and display tasks
- **Counter & Like Button**: State and event handling demo
- **Lottery Game**: Generate tickets and check winning sum
- **Product Price**: Display products with old price cut
- **Form**: Controlled form inputs demo
